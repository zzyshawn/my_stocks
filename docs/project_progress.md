## 🎯 业务目标与里程碑跟踪

- [x] 多时间周期的数据爬取与聚合（实现日线、30分钟、5分钟的独立切片存储）。
- [x] 涵盖大部分本土因子的底层逻辑与指标库开发。
- [x] 跑通单个标的的验证流。

### 🏆 Phase 2 - 工业级量化中台跨越 (已达成)
- [x] 告别弱相关的独立遍历，引入 **原生 QLib 引擎**，统一管理特征集 (`data.qlib_dir`)。
- [x] 分治系统：`qlib_dump` 负责喂数据、`factors` 负责加工数据流水线。
- [x] **LightGBM / AI 核心引擎** 集成落地。支持每天动态打分并依据排名执行模拟回测。

### 🚀 Phase 3 - 实盘决策与消息流 (正在 / 即将推进)
- [ ] **决策与盘前清单聚合**：每日把回测与最新的 QLib 预测单组合输出，形成肉眼可读的买前“监控看板”。
- [x] **通信通知对接**：完成飞书消息机制对接，实现核心量化预警的分发。
- [ ] 实盘与模拟交易网关对接（可能采用外部组件或特定证券 API）。
