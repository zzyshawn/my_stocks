# 决策模块需求

## 1. 功能描述
综合技术分析结果，生成买卖决策建议。

## 2. 功能需求

| 编号 | 需求项 | 优先级 | 描述 |
|------|--------|--------|------|
| R5.1 | 信号生成 | P0 | 根据技术指标生成买入/卖出/持有信号 |
| R5.2 | 信号强度 | P1 | 计算信号强度评分（1-10分） |
| R5.3 | 风险评估 | P1 | 评估当前持仓风险等级 |
| R5.4 | 止损止盈 | P1 | 计算建议止损止盈价位 |
| R5.5 | 仓位建议 | P2 | 根据风险等级给出仓位建议 |
| R5.6 | 多因子模型 | P2 | 支持多因子综合评分模型 |
| R5.7 | 自定义策略 | P2 | 支持用户自定义决策策略 |
| R5.8 | 决策日志 | P1 | 记录历史决策及其结果 |

## 3. 决策信号等级

| 信号 | 描述 | 建议操作 |
|------|------|----------|
| 强烈买入 | 多项指标共振，信号强度>=8 | 积极建仓 |
| 买入 | 指标偏多，信号强度6-7 | 适度建仓 |
| 持有 | 指标中性，信号强度4-5 | 观望持有 |
| 卖出 | 指标偏空，信号强度2-3 | 减仓 |
| 强烈卖出 | 多项指标恶化，信号强度<=1 | 清仓 |

## 4. 模块结构

```
src/decision/
├── __init__.py
├── signals.py          # 信号生成
└── strategies.py       # 策略管理
```

## 5. 配置参数

决策参数可在配置文件中自定义：

```yaml
decision:
  # 信号强度阈值
  signal_threshold:
    strong_buy: 8
    buy: 6
    hold: 4
    sell: 2
    strong_sell: 1

  # 止损止盈比例
  stop_loss_pct: 0.05   # 止损 5%
  take_profit_pct: 0.15 # 止盈 15%
```
